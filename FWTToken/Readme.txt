*************************************
To run the Project:
*************************************

Tools Required
1. Install Ganache
2. Install Node
3. Install Truffle 

*************************************
!!!Important - Run below command 
*************************************
npm i

*************************************
To test contracts
*************************************

1. Open Ganache change the RPC server details according in truffle.config.js
2. truffle compile
3. from project's directory in terminal 
4. truffle compile
5. truffle test test/<filenameinsdetestdirectory.js>

*************************************
To Migrate 
*************************************

1. truffle compile
2. truffle migrate

*************************************
To run Web interface
*************************************

1. truffle compile 
2. truffle migrate
3. npm run dev
